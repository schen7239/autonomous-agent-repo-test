# Golang Project Setup\n\n## Getting Started\n\n### Prerequisites\n- Go 1.20+\n- Docker\n\n### Installation\n\n1. Clone the repository:\n   \`\`\`bash\ngit clone https://github.com/schen7239/autonomous-agent-repo-test.git\n\`\`\`\n2. Navigate to the project directory:\n   \`\`\`bash\ncd autonomous-agent-repo-test\n\`\`\`\n3. Install dependencies:\n   \`\`\`bash\ngo mod tidy\n\`\`\`\n\n### Running the Project\n\n1. Start the application:\n   \`\`\`bash\ngo run main.go\n\`\`\`\n\n### Testing\n\n1. Run tests:\n   \`\`\`bash\ngo test ./...\n\`\`\`\n\n## Contributing\n\n1. Fork the repository.\n2.
